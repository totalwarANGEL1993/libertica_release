RestartQuest (_QuestName, _NoMessage)
Startet einen Quest neu.

Quests müssen beendet sein, um erneut gestartet zu werden. Entweder müssen alle Trigger erfolgreich sein oder der Quest muss manuell ausgelöst werden.

Parameters:

  • _QuestName string Name des Quests
  • _NoMessage boolean Debug-Nachricht deaktivieren

Returns:

  1. integer ID Neue Quest-ID
  2. table Data Quest-Instanz
