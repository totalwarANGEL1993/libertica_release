Module module.quest.questjornal_api

Ermöglicht das Hinzufügen von Notizen zu einem Quest.

Notizen zu einem Auftrag werden in einem Textfenster angezeigt. Für Quests mit Notizen wird ein neuer Button angezeigt, mit dem das Textfenster geöffnet werden kann. Notizen können in normale (blau) und erhöhte (rot) Wichtigkeit eingeteilt werden. Zusätzlich kannst du dem Spieler erlauben, sich selbst Notizen zu machen. Die Notizen des Spielers haben eine höhere Priorität gegenüber allen anderen Notizen und sich violet hervorgehoben.

Functions

ShowJournalForQuest (_Quest, _Flag) Aktiviert oder deaktiviert das Journal oder einem bestimmtem Quest.
AllowNotesForQuest (_Quest, _Flag) Ermöglicht es dem Spieler, Notizen im Journal zu machen.
CreateJournalEntry (_Text) Erstellt einen neuen Journaleintrag und gibt die ID zurück.
AlterJournalEntry (_ID, _Text) Aktualisiert den Journaleintrag mit der ID.
HighlightJournalEntry (_ID, _Important) Markiert einen Journaleintrag als wichtig und hebt ihn hervor.
DeleteJournalEntry (_ID) Löscht einen Eintrag.
RestoreJournalEntry (_ID) Stellt einen gelöschten Eintrag wieder her.
AddJournalEntryToQuest (_ID, _Quest) Fügt einen Eintrag dem Journal des Quest hinzu.
RemoveJournalEntryFromQuest (_ID, _Quest) Entfernt einen Eintrag aus dem Journal des Quest.
