Module module.information.requester_api

Bietet Dialog- und Textfenster-Funktionen.

Info-Dialog

Ein einfaches Fenster für die Ausgabe einer Information. Das Fenster muss mit OK bestätigt werden und kann anschließend eine Funktion ausführen.

Rückfrage-Dialog

Ein einfaches Fenster für eine binäre Entscheidung. Das Fenster kann mit Ja/Nein bzw. OK/Cancel geschlossen werden. In der Callback-Funktion kann anhand der gewählten Option unterschieden werden.

Auswahl-Dialog

Ein Fenster mit einem Text und einem Dropdown Menü. Es kann eine Option im Menü ausgewählt werden. Danach wird mit OK bestätigt. Im Callback kann anhand der gewählten Option unterschieden werden.

Sprachwahl-Dialog

Ein spezieller Auswahl-Dialog, der die Sprache für User-Texte einstellen kann. Verwende diesen Dialog, wenn deine Map mehrsprachig ist. Wird die Sprache gewählt, wird ein Event ausgelöst.

Textfenster

Eine modifizierte Version des Chatlog für die Ausgabe eines größeren Text. Wird das Fenster geschlossen, kann eine Callback-Funktion ausgeführt werden. Optional kann ein Button unter dem Text angezeigt werden. Der Button hat eine eigene Callback-Funktion.

Functions

DialogInfoBox (_PlayerID, _Title, _Text, _Action) Öffnet einen einfachen Dialog.

DialogLanguageSelection (_PlayerID) Zeigt die Sprachauswahl für einen Spieler an.

TextWindow (_Caption, _Content, _PlayerID)
Öffnet ein großes Textfenster mit dem bereitgestellten Text.

Example:

 local Text = "Lorem ipsum dolor sit amet, consetetur".."..
              " sadipscing elitr, sed diam nonumy eirmod"..

  • _Caption string Fenstertitel
  • _Content string Fensterinhalt

Parameters:

  • _PlayerID integer (Optional) ID des Empfängers
  • _Title any Titel des Fensters
  • _Text any Text des Fensters
  • _Action function Aktionsfunktion
DialogRequestBox (_PlayerID, _Title, _Text, _Action, _OkCancel)
Öffnet einen Dialog mit einer Ja/Nein-Option.

Example:

 function YesNoAction(_Ja, _PlayerID)
     if _Ja then
         GUI.AddNote("'Ja' wurde angeklickt!");
     end
 end
 DialogRequestBox("Frage", "Möchtest du das wirklich tun?", YesNoAction, false);

Parameters:

  • _PlayerID integer (Optional) ID des Empfängers
  • _Title any Titel des Fensters
  • _Text any Text des Fensters
  • _Action function Aktionsfunktion
  • _OkCancel boolean Verwenden Sie Okay/Abbrechen für Schaltflächen
DialogSelectBox (_PlayerID, _Title, _Text, _Action, _List)
Öffnet einen Dialog mit einer Optionsbox.

Example:

 function OptionsAction(_Idx, _PlayerID)
     GUI.AddNote(_Idx.. " wurde ausgewählt!");
 end
 local List = {"Option A", "Option B", "Option C"};
 DialogSelectBox("Auswahl", "Wähle eine Option!", OptionsAction, List);

Parameters:

  • _PlayerID integer (Optional) ID des Empfängers
  • _Title any Titel des Fensters
  • _Text any Text des Fensters
  • _Action function Aktionsfunktion
  • _List table Liste der Optionen
