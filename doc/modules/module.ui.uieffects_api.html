ActivateImageScreen (_PlayerID, _Image, _Red, _Green, _Blue, _Alpha)
Zeigt eine Vollbildgrafik über der Spielwelt und unterhalb der Benutzeroberfläche an.

Die Grafik muss im 16:9-Format geliefert werden. Für 4:3-Auflösungen wird die Grafik links und rechts zentriert, um das Format anzupassen.

Parameters:

  • _PlayerID integer ID des Spielers
  • _Image string Pfad zur Grafik
  • _Red integer? (Optional) Rotwert (Standard: 255)
  • _Green integer? (Optional) Grünwert (Standard: 255)
  • _Blue integer? (Optional) Blauwert (Standard: 255)
  • _Alpha integer? (Optional) Alphawert (Standard: 255)
