Module module.trade.trade_api

Dieses Modul erlaubt Kauf und Verkauf zu manipulieren.

Der Aufbau von Angeboten kann untersucht und manipuliert werden. Du kannst Einzelne Angebote löschen oder die Menge bearbeiten. Die Berechnung von Kauf- und Verkaufspreise kann angepasst werden. So kannst du auch Elias Fähigkeit verstärken oder abschwächen.
