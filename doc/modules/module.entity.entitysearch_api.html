    CommenceEntitySearch (_Filter)
    Sucht Entitäten anhand des übergebenen Filters.

    Ein Filter ist eine Funktion, die für jede gefundene Entität prüft, ob sie zur Ergebnismenge gehört oder nicht. Um die Filterung durchzuführen, wird der Name angegeben unter dem die Funktion gespeichert wurde.

    Example:

    Dieser Filter akzeptiert alle Entitäten von Spieler 1. 
     -- Filter erzeugen
 CreateSearchFilter("MyFilter", function(_ID)
     if Logic.EntityGetPlayer(_ID) ~= 1 then
         return false;
     end
     return true;
 end);
 -- Filter ausführen
 local Results = CommenceEntitySearch("MyFilter");

    Parameters:

    • _Filter any

    Returns:

      table Results Liste mit Entitäten
