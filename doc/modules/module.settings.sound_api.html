Module module.settings.sound_api

Dieses Modul ermöglicht die Steuerung des Sounds.

Funktionen

  • Starten/Stoppen von Musik-Playlists
  • Starten/Stoppen von Sprachwiedergabe
  • Starten/Stoppen von Soundeffekten
  • Ändern der Soundkonfiguration

    • Aufbau Playlist

     <?xml version="1.0" encoding="utf-8"?>
 <PlayList>
  <PlayListEntry>
    <FileName>Music\some_music_file.mp3</FileName>
    <Type>Loop</Type>
    <Chance>10</Chance> <!-- optional -->
  </PlayListEntry>
  <!-- Add entries here -->
 </PlayList>

    Functions

    StartEventPlaylist (_Playlist, _PlayerID)
    Startet eine Ereignis-Playlist.

    Es ist möglich, benutzerdefinierte Dateien in das Spiel zu importieren, indem sie in das Kartenarchiv gespeichert werden. Eine Playlist benötigt eine XML-Datei, die sie definiert, und die Dateien mit der Musik.

    Um Probleme mit Musikdateien zu vermeiden, ist es am besten, allen Dateien einen eindeutigen Namen zu geben. Zum Beispiel finden Sie für alle Dateien aus dem Kartenname einen kurzen Präfix. Wenn mehrere Karten mit denselben Dateien innerhalb der Playlist oder des Musikverzeichnisses vorhanden sind, werden Dateien überschrieben!

    Dateistruktur für benutzerdefinierte Playlists:

     map_xyz.s6xmap.unpacked
 |-- music/*
 |-- config/sound/*
 |-- maps/externalmap/map_xyz/*
 |-- ...

    Playlist-Einträge können mit Loop als Typ wiederholt oder einmal abgespielt werden mit Normal als Typ.

    Parameters:

    • _Playlist string Name der Playlist
    • _PlayerID integer ID des Spielers
    StopEventPlaylist (_Playlist, _PlayerID)
    Stoppt eine Ereignis-Playlist.

    Parameters:

    • _Playlist string Name der Playlist
    • _PlayerID integer ID des Spielers
    Play2DSound (_Sound, _PlayerID)

    Es ist möglich, benutzerdefinierte Dateien in das Spiel zu importieren, indem sie in das Kartenarchiv gespeichert werden. Um Namensprobleme zu vermeiden, geben Sie allen Dateien einen eindeutigen Namen.

    Dateistruktur für benutzerdefinierte Sounds:

 |-- sounds/high/ui/*
 |-- sounds/low/ui/*
 |-- maps/externalmap/map_xyz/*
 |-- ...

    Parameters:

    • _Sound string Pfad zum Sound
    • _PlayerID integer ID des Spielers
