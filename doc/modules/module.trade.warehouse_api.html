Module module.trade.warehouse_api

Ermöglicht die Erstellung von Lagern.

Lager sind modifizierte Handelsposten, an denen der Spieler Waren ohne Beteiligung eines KI-Spielers kaufen kann. Es können jedoch keine Waren an das Lager verkauft werden. Die Bezahlung kann auf jeden beliebigen Ressourcentyp eingestellt werden.

Functions

CreateWarehouse (_Data)
Definiert eine Baustelle für einen Handelsposten als Lager.

Fields _Data:

  • ScriptName: string Skriptname der Baustelle
  • Offers: table Liste der Angebote (max. 6 sichtbare Angebote)

    • Fields _Data.Offers:

  • Amount: integer Menge an Angeboten
  • GoodType: integer Typ der Ware des Angebot
  • GoodAmount: integer Menge der Ware pro Angebot
  • PaymentType: (optional) integer Bezahlart der Kosten (Default: Goods.G_Gold)
  • BasePrice: (optional) integer Kosten
  • Refresh: (optional) integer Regenerationszeit pro Angebot

    • Ecample:

     CreateWarehouse {
     ScriptName       = "TP3",
     Offers           = {
         -- Ressourcenangebot
         {Amount      = 3,
          GoodType    = Goods.G_Iron,
          BasePrice   = 80},
         -- Produktangebot
         {Amount      = 3,
          GoodType    = Goods.G_Sausage,
          BasePrice   = 150},
         -- Luxusangebot
         {Amount      = 3,
          GoodType    = Goods.G_Gems,
          GoodAmount  = 27,
          BasePrice   = 300},
         -- Unterhaltungsangebot
         {GoodType    = Entities.U_Entertainer_NA_FireEater,
          BasePrice   = 250,
          Refresh     = 500},
         -- Söldnerangebot
         {Amount      = 3,
          GoodType    = Entities.U_MilitaryBandit_Melee_ME,
          PaymentType = Goods.G_Iron,
          BasePrice   = 3},
         -- Söldnerangebot
         {Amount      = 3,
          GoodType    = Entities.U_CatapultCart,
          BasePrice   = 1000},
     },
 };

    Parameters:

    • _Data table Konfiguration des Lagers
