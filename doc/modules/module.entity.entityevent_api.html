Module module.entity.entityevent_api

Fügt nützliche Berichte zu Entitäten hinzu.

Functions

ThiefDisableStorehouseEffect (_Flag) Aktiviert/Deaktiviert die Sabotage eines Lagerhauses durch Diebe.
ThiefDisableCathedralEffect (_Flag) Aktiviert/Deaktiviert Die Sabotage einer Kirche durch Diebe.
ThiefDisableCisternEffect (_Flag) Aktiviert/Deaktiviert Die Sabotage von Brunnen durch Diebe.
