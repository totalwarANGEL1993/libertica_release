Module module.information.cutscenesystem_api

Ermöglicht das Definieren von Zwischensequenzen.

Zwischensequenzen sind XML-definierte Kamerabewegungen, die von der Spiel-Engine wiedergegeben werden können. Zwischensequenzen zeichnen sich durch flüssige Kamerabewegungen aus.

Functions

NewCutscene (_Name, _PlayerID) Initialisiert den Builder für eine Cutscene.
IsCutsceneActive (_PlayerID) Überprüft, ob eine Zwischensequenz aktiv ist.
StartCutscene (_Cutscene, _Name, _PlayerID) Startet eine Zwischensequenz.
AddCutscenePages (_Cutscene) Bereitet die Zwischensequenz vor und gibt die Seitenfunktion zurück.
AP (_Data) Erstellt eine Seite.


Functions

NewCutscene (_Name, _PlayerID)
Initialisiert den Builder für eine Cutscene.

Functions CutsceneBuilder:

  • SetName(_Name): Setzt den Namen der Cutscene.
  • SetPlayer(_Player): Setzt den empfangenden Spieler der Cutscene.
  • UseBigBars(_Flag): Verwendet die breiten Briefing Bars. (Aus ästhetischen Gründen nicht zu empfehlen.)
  • UseGlobalImmortality(_Flag): Während der Cutscene sind alle Entities unverwundbar.
  • SetHideNotes(_Flag): Versteckt das Notes Window wärhend der Cutscene.
  • SetEnableSky(_Flag): Zeigt den Himmel während der Cutscene an.
  • SetEnableFoW(_Flag): Zeigt den Nebel des Krieges während der Cutscene an.
  • SetEnableBorderPins(_Flag): Zeigt die Grenzsteine während der Cutscene an.
  • SetOnBegin(_OnBegin): Funktion, die beim Start der Cutscene ausgeführt wird.
  • SetOnFinish(_OnFinish): Funktion, die beim Ende der Cutscene ausgeführt wird.
  • BeginFlight(): Eröffnet den FlightBuilder.
  • Start(): Startet die Cutscene.

    • Functions FlightBuilder:

  • SetSpeech(_Speech): Setzt den Pfad zur Voice Line.
  • SetTitle(_Title): Setzt den anzuzeigenden Titel der Page.
  • SetText(_Text): Setzt den anzuzeigenden Text der Page.
  • SetFadeIn(_Time): Blendet von Schwarz ein.
  • SetFadeOut(_Time): Blendet zu Schwarz aus.
  • SetFaderAlpha(_Opacity): Setzt den Alphawert der Fadermaske.
  • UseBigBars(_Flag): Verwendet die breiten Cutscene Bars auf dieser Seite. (Aus ästhetischen Gründen nicht zu empfehlen.)
  • UseSkipping(_Flag): Die Seite kann übersprungen werden.
  • SetFarClipPlane(_Distance): Setzt die maximale Sichtweite. Achtung: Kann auf schwachen Systemen ruckeln.
  • EndFlight(): Beendet den FlightBuilder und kehrt zum CutsceneBuilder zurück.

    • Example:

     NewCutscene("TestCutscene", 1)
     :SetEnableBorderPins(true)
     :SetEnableSky(true)
     :BeginFlight()
         :SetFlight("c01")
         :SetTitle("Flight 1")
         :SetText("This is the shown text.")
         :SetFadeIn(3)
     :EndFlight()
     :BeginFlight()
         :SetFlight("c02")
         :SetTitle("Flight 2")
         :SetText("This is the shown text.")
     :EndFlight()
     :BeginFlight()
         :SetFlight("c03")
         :SetTitle("Flight 3")
         :SetText("This is the shown text.")
         :SetFadeOut(3)
     :EndFlight()
     :Start();

    Parameters:

    • _Name string Name der Cutscene
    • _PlayerID integer Spieler-ID des Empfängers

    Returns:

      table BriefingBuilder Builder der Cutscene

    StartCutscene (_Cutscene, _Name, _PlayerID)
    Startet eine Zwischensequenz.

    Achtung: Diese Funktion ist veraltet und kann nur mit der deklarativen API benutzt werden.

    Fields _Cutscene:

  • Starting: (optional) function Funktion, die beim Starten der Einleitung aufgerufen wird
  • Finished: (optional) function Funktion, die beim Beenden der Einleitung aufgerufen wird
  • EnableGlobalImmortality: (optional) boolean Während Einleitungen sind alle Entitäten unverwundbar
  • EnableSky: (optional) boolean Zeigt den Himmel während der Einleitung an
  • EnableFoW: (optional) boolean Zeigt den Nebel des Krieges während der Einleitung an
  • EnableBorderPins: (optional) boolean Zeigt die Randnadeln während der Einleitung an
  • HideNotes: (optional) boolean Nachrichten nicht anzeigen

    • Example:

     function Cutscene1(_Name, _PlayerID)
     local Cutscene = {};
     local AP = API.AddCutscenePages(Cutscene);
     -- Seiten
     Cutscene.Starting = function(_Data)
     end
     Cutscene.Finished = function(_Data)
     end
     API.StartCutscene(Cutscene, _Name, _PlayerID);
 end

    Parameters:

    • _Cutscene table Zwischensequenz-Tabelle
    • _Name string Name der Zwischensequenz
    • _PlayerID integer Spieler-ID des Empfängers
    AddCutscenePages (_Cutscene)
    Bereitet die Zwischensequenz vor und gibt die Seitenfunktion zurück.

    Achtung: Diese Funktion ist veraltet und kann nur mit der deklarativen API benutzt werden.

    Muss aufgerufen werden, bevor Seiten hinzugefügt werden.

    Parameters:

    • _Cutscene table Zwischensequenz-Tabelle

    Returns:

      function AP Seitenfunktion
    AP (_Data)
    Erstellt eine Seite.

    Achtung: Diese Funktion ist veraltet und kann nur mit der deklarativen API benutzt werden.

    Fields _Data:

  • Flight: string Name der Flug-XML (ohne .cs)
  • Title: (optional) any Angezeigter Seitentitel (String oder Language Table)
  • Text: (optional) any Angezeigter Seitentext (String oder Language Table)
  • Speech: (optional) string Pfad zum Voiceover (MP3-Datei)
  • Action: (optional) function Funktion, die beim Anzeigen der Seite aufgerufen wird
  • FarClipPlane: (optional) boolean Render-Entfernung
  • FadeIn: (optional) float Dauer des Einblendens aus Schwarz
  • FadeOut: (optional) float Dauer des Ausblendens nach Schwarz
  • FaderAlpha: (optional) float Maskenalpha
  • DisableSkipping: (optional) boolean Erlauben/Verbieten des Überspringens von Seiten
  • BarOpacity: (optional) float Deckkraft der Leisten
  • BigBars: (optional) boolean Verwendung großer Leisten

    • Example:

     AP {
     Flight       = "c02",
     FarClipPlane = 45000,
     Title        = "Titel",
     Text         = "Text des Flugs.",
 };

    Parameters:

    • _Data table Seitendaten

    Returns:

      table Page Erzeugte Seite
