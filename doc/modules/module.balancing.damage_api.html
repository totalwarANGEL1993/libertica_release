Module module.balancing.damage_api

Ermöglicht die Veränderung der Schadenswerte von Einheiten.

Achtung: Die Funktionen MakeVulnerable und MakeInvulnerable wurden überschrieben. Die Funktion Logic.SetEntityInvulnerabilityFlag wird stattdessen intern benutzt und darf nicht mehr verwendet werden!

Funktionen:

  • Besseres Balancing für Bogenschützen
  • Nahe Mauerkatapulte schwächen sich gegenseitig (gilt nicht für KI)
  • Anpassen der Kampfkraft von Einheiten über Lua
  • Zusätzliche Rüstung für Einheiten vergeben
  • Anpassen des Own Territoy Bonus
  • Anpassen des Height Modifier

    • Achtung: Diese Funktionen werden sofort aktiv, wenn das Modul geladen wird!

    Functions

    SetEntityTypeDamage (_Type, _Damage, ...)
    Setzt den Schaden für einen Entitätstypen.

    Optional können Feindkategorien angegeben werden, für die der Schaden gilt.

    Der angegebene Schaden zu einer Liste von Kategorien ersetzt den Basischaden des Entitätstypen. Der finale Schaden Ergibt sich aus Moral, Höhenbonus und Territorienbonus.

    Parameters:

    • _Type integer Entitätstyp
    • _Damage integer Höhe des Schaden
    • ... integer? Optionale Feindkategorien
    SetEntityNameDamage (_Name, _Damage, ...)
    Setzt den Schaden für eine benannte Entität.

    Optional können Feindkategorien angegeben werden, für die der Schaden gilt.

    Der angegebene Schaden zu einer Liste von Kategorien ersetzt den Basischaden der Entität. Der finale Schaden Ergibt sich aus Moral, Höhenbonus und Territorienbonus.

    Parameters:

    • _Name string Scriptname der Entität
    • _Damage integer Höhe des Schaden
    • ... integer? Optionale Feindkategorien
    SetEntityTypeArmor (_Type, _Armor)
    Setzt Die Rüstung für einen Entitätstypen.

    Der Rüstungswert wird nach der eigentlichen Schadensberechnung vom übrigen Schaden abgezogen. Der Schaden kann niemals kleiner als 1 werden.

    Parameters:

    • _Type integer Entitätstyp
    • _Armor integer Stärke der Rüstung
    SetEntityNameArmor (_Name, _Armor)
    Setzt die Rüstung für eine benannte Entität.

    Der Rüstungswert wird nach der eigentlichen Schadensberechnung vom übrigen Schaden abgezogen. Der Schaden kann niemals kleiner als 1 werden.

    Parameters:

    • _Name string Scriptname der Entität
    • _Armor integer Stärke der Rüstung
    SetTerritoryBonus (_PlayerID, _Bonus)
    Setzt den Bonusfaktor für den Schaden bei Kämpfen auf eigenem Gebiet.

    Der angegebene Faktor wird mit dem eigentlichen Faktor multipliziert. Für 0.5 bedeutet dies, der Territorienbonus wird halbiert. Bei der Berechnung wird der Territorienbonus zur Moral dazu addiert.

    Parameters:

    • _PlayerID integer ID des Spielers
    • _Bonus number Faktor
    SetHeightModifier (_PlayerID, _Bonus)
    Setzt den Bonusfaktor für den Schaden bei Kämpfen von erhöhter Position.

    Der angegebene Faktor wird mit dem eigentlichen Faktor multipliziert. Für 0.5 bedeutet dies, der Höhenbonus wird halbiert. Bei der Berechnung wird die Summe von Moral und Territorienbonus mit dem Höhenbonus multipliziert.

    Parameters:

    • _PlayerID integer ID des Spielers
    • _Bonus number Faktor
