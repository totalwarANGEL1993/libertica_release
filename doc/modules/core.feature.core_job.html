Module core.feature.core_job

Ermöglicht das Starten von Jobs verschiedener Ereignistypen und deren Steuerung.

Auch einfache Verzögerungen werden bereitgestellt.

Functions

RequestJobByEventType (_EventType, _Function, ...) Fordert einen Job des übergebenen Ereignistyps an.
RequestJob (_Function, ...) Fordert einen Job an, der jede Sekunde ausgelöst wird.
RequestHiResJob (_Function, ...) Fordert einen Job an, der jede Runde ausgelöst wird.
RequestDelay (_Waittime, _Function, ...) Fordert eine verzögerte Aktion an, verzögert um Sekunden.
RequestHiResDelay (_Waittime, _Function, ...) Fordert eine verzögerte Aktion an, verzögert um Runden.
RequestRealTimeDelay (_Waittime, _Function, ...) Fordert eine verzögerte Aktion an, verzögert um Echtzeit-Sekunden.
StopJob (_JobID) Beendet einen Job.
IsJobRunning (_JobID) Gibt zurück, ob der Job ausgeführt wird.
ResumeJob (_JobID) Setzt einen pausierten Job fort.
YieldJob (_JobID) Pausiert einen laufenden Job.
GetSecondsRealTime () Gibt die seit dem Spielstart vergangenen Echtzeit-Sekunden zurück.
