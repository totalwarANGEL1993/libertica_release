Module module.settings.camera_api

Funktionen zur Manipulation der RTS-Kamera.

Functions

SetRenderDistance (_View) Ändert die maximale Darstellungsreichweite, bis etwas abgeschnitten wird.
ResetRenderDistance () Setzt das Clipping auf seine Standardwerte zurück.
ActivateBorderScroll (_PlayerID) Aktiviert den Bildlauf am Rand des Bildschirms und lockert die Kamerabefestigung.
DeactivateBorderScroll (_PlayerID, _Position) Deaktiviert den Bildlauf am Rand des Bildschirms und fixiert die Kamera.
ActivateExtendedZoom (_PlayerID) Aktiviert den erweiterten Zoom für einen Spieler oder für alle Spieler.
DeactivateExtendedZoom (_PlayerID) Deaktiviert den erweiterten Zoom für einen Spieler oder für alle Spieler.
SetNormalZoomProps (_Limit) Setzt das Zoom Limit für den normalen Zoom.
SetExtendedZoomProps (_Limit) Setzt das Zoom Limit für den erweiterten Zoom.
FocusCameraOnKnight (_PlayerID, _Rotation, _ZoomFactor) Fokussiert eine Kamera auf den Ritter des Spielers, sperrt die Kamera jedoch nicht.
FocusCameraOnEntity (_Entity, _Rotation, _ZoomFactor) Fokussiert eine Kamera auf eine Entität, sperrt die Kamera jedoch nicht.
