Module module.io.io_api

Verbessert interaktive Objekte.

Fast alle Entitäten können als interaktives Objekt verwendet werden, nicht nur die, die dafür vorgesehen sind. Ein Objekt wird durch eine Tabelle beschrieben und (fast) alle Schlüssel sind optional.

Functions

SetupObject (_Description) Fügt einem Objekt eine Interaktion hinzu.
DisposeObject (_ScriptName) Entfernt die Interaktion vom Objekt.
ResetObject (_ScriptName) Setzt das interaktive Objekt zurück.
InteractiveObjectAddCustomName (_Key, _Text) Ändert den Namen des Objekts in der 2D-Oberfläche.
InteractiveObjectDeleteCustomName (_Key) Entfernt die Änderungen am Objektnamen.
AllowActivateIronMines (_PlayerID, _Allowed) Ermöglicht oder verbietet das Auffüllen von Eisenerzminen.
RequireTitleToRefilIronMines (_Title) Legt den erforderlichen Titel zum Auffüllen von Eisenerzminen fest.
AllowActivateStoneMines (_PlayerID, _Allowed) Ermöglicht oder verbietet das Auffüllen von Steinbrüchen.
RequireTitleToRefilStoneMines (_Title) Legt den erforderlichen Titel zum Auffüllen von Steinbrüchen fest.
AllowActivateCisterns (_PlayerID, _Allowed) Ermöglicht oder verbietet das Auffüllen von Zisternen.
RequireTitleToRefilCisterns (_Title) Legt den erforderlichen Titel zum Auffüllen von Zisternen fest.
AllowActivateTradepost (_PlayerID, _Allowed) Ermöglicht oder verbietet den Bau von Handelsposten.
RequireTitleToBuildTradeposts (_Title) Legt den erforderlichen Titel zum Bau von Handelsposten fest.
InteractiveObjectActivate (_ScriptName, _State, ...) Aktiviert ein interaktives Objekt.
InteractiveObjectDeactivate (_ScriptName, ...) Deaktiviert ein interaktives Objekt.

Fields

Report.ObjectClicked Der Spieler hat auf die Interaktions-Schaltfläche geklickt.
Report.ObjectInteraction Die Interaktion des Objekts war erfolgreich.
Report.ObjectReset Die Interaktion wird vom Objekt gelöscht.
Report.ObjectDelete Der Zustand eines Objekts wurde zurückgesetzt.


Functions

SetupObject (_Description)
Fügt einem Objekt eine Interaktion hinzu.

Fields _Description:

  • ScriptName - Skriptname des Objekts
  • Texture - (Optional) Tabelle mit Koordinaten
  • Title - (Optional) Titel des Tooltipps
  • Text - (Optional) Text des Tooltipps
  • Distance - (Optional) Aktivierungsabstand
  • Player - (Optional) Liste der Spieler
  • Waittime - (Optional) Aktivierungs-Wartezeit
  • Replacement - (Optional) Typ, mit dem ersetzt wird
  • Costs - (Optional) Aktivierungskosten-Tabelle
  • Reward - (Optional) Aktivierungsbelohnungstabelle
  • State - (Optional) Aktivierungsverhalten
  • Condition - (Optional) Aktivierungsbedingungsfunktion
  • ConditionInfo - (Optional) Text bei Bedingungsfehler
  • Action - (Optional) Aktivierungs-Callback-Funktion
  • RewardResourceCartType - (Optional) Typ des Belohnungsressourcenkarrens
  • RewardGoldCartType - (Optional) Typ des Belohnungsgoldkarrens
  • CostResourceCartType - (Optional) Typ des Kostenressourcenkarrens
  • CostGoldCartType - (Optional) Typ des Kostengoldkarrens

    • Example:

     -- Erstellt ein einfaches Objekt
 SetupObject {
     Name     = "hut",
     Distance = 1500,
     Reward   = {Goods.G_Gold, 1000},
 };

    Parameters:

    • _Description table Objektbeschreibung

    Returns:

      table? Data Objektbeschreibung

    Ändert den Namen des Objekts in der 2D-Oberfläche.

     de = "Hafenkran",

 });
