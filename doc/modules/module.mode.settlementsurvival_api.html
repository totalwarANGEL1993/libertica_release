Module module.mode.settlementsurvival_api

Implementiert Überlebensaspekte von Siedlungen im Spiel.

Siedler (und Nutztiere) können nun an verschiedenen Ursachen erkranken und möglicherweise sterben. Wenn ein Siedler stirbt, bleibt er weiterhin mit dem Gebäude verbunden, sodass kein anderer Siedler ihn ersetzen kann. Nach einiger Zeit wird ein neuer Siedler eintreffen. Außerdem ist das Bedürfnis nach Kleidung auch für Sammler aktiv.

Dinge, die Gebäude entzünden:

  • Hohe Temperatur

    • Dinge, die Siedler krank machen:

  • Unzureichende Hygiene
  • Unzureichende Unterhaltung
  • Unzureichendes Brennholz

    • Dinge, die Siedler töten:

  • Krankheit (5%ige Chance pro Periode)
  • Hunger (5%ige Chance pro Periode)

    • Dinge, die Tiere töten:

  • Krankheit (5%ige Chance pro Periode)
