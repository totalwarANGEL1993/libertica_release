Module core.core_api

Die Funktionalotäten des Basis-Modul.

Scripting Values:

  • CONST_SCRIPTING_VALUES.Destination.X: integer X-Koordinate des Bewegungsziels
  • CONST_SCRIPTING_VALUES.Destination.Y: integer Y-Koordinate des Bewegungsziels
  • CONST_SCRIPTING_VALUES.Health: integer Gesundheit der Entität
  • CONST_SCRIPTING_VALUES.Player: integer Spieler-ID der Entität
  • CONST_SCRIPTING_VALUES.Size: integer Skalierung der Entität
  • CONST_SCRIPTING_VALUES.Visible: integer Ist die Entität sichtbar (= 801280)
  • CONST_SCRIPTING_VALUES.NPC: integer Art des NPCs (> 0)

    • Console Commands:

    In der Konsole können spezielle Komandos eingegeben werden. Eingeklammerte Angaben sind dabei optional.
  • restartmap: Startet die Map sofort neu
  • clear: Entfernt alle Nachrichten vom Bildschirm
  • version: Zeigt die aktuelle Version von Libertica an
  • > any code: Führt beliebigen Code in der globalen Umgebung aus
  • >> any code: Führt beliebigen Code in der lokalen Umgebung aus
  • < path: Läd ein Lua-File in die globale Umgebung
  • << path: Läd ein Lua-File in die lokale Umgebung
